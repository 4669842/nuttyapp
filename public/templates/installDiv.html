Supported OS/Browser: Linux/Mac with Chrome-29+
Your OS/Browser: [[Compatibility.browser.OS]]/[[Compatibility.browser.browser]]-[[Compatibility.browser.version]] Installation steps:
  • Install Extension
  • Install tmux
    • Ubuntu: sudo apt-get install tmux
    • Fedora: sudo yum install tmux
    • MacOSX: sudo brew install tmux
  • Run the following command in terminal:
    [[installcmd]]
After installation click icon beside the address bar.
