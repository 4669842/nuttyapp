Installation (3 steps):
  1. Install Extension
    After installing the extension you will see icon beside the address bar.
  2. Install tmux
    • Ubuntu: sudo apt-get install tmux
    • Fedora: sudo yum install tmux
    • MacOSX: sudo brew install tmux
      (install brew from here)
  3. Run the following command in terminal: 
    [[installcmd]]
After step-3 click icon beside the address bar to start terminal in the browser, it will also give you a unique URL which you can share with a remote user.
