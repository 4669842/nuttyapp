Installation steps:
  • Install Extension
  • Install tmux
    • Ubuntu: sudo apt-get install tmux
    • Fedora: sudo yum install tmux
    • MacOSX: sudo brew install tmux
  • Run the following command in terminal:
    [[installcmd]]
After installation click icon beside the address bar.

Optionally you can run your own nutty server too as nutty is Open Source under GPL.

Deploy nutty server meteor.com:

  • $ curl https://install.meteor.com/ | sh
  • $ sudo npm install -g meteorite
  • $ git clone https://github.com/krishnasrinivas/nuttyapp.git
  • $ cd nuttyapp
  • $ mrt install
  • $ meteor deploy <somename>.meteor.com

Deploy nutty server on a cloud server like EC2:

  • Install MongoDB
  • $ curl https://install.meteor.com/ | sh
  • $ sudo npm install -g meteorite
  • $ git clone https://github.com/krishnasrinivas/nuttyapp.git
  • $ cd nuttyapp
  • $ mrt install
  • $ meteor bundle ../bundle.tgz
  • $ cd ..
  • $ tar xzvf bundle.tgz
  • $ export MONGO_URL=mongodb://localhost/nuttyapp
  • $ node bundle/main.js

Hosting nutty server on Heroku (using free dyno):

  • Signup at mongohq.com, they offer free-tier
  • git clone https://github.com/krishnasrinivas/nuttyapp.git nuttyapp-heroku
  • cd nuttyapp-heroku
  • heroku apps:create <nuttyserver>
  • heroku config:add BUILDPACK_URL=https://github.com/krishnasrinivas/heroku-buildpack-meteorite.git
  • heroku config:add MONGO_URL=<mogodb URL from mogohq.com>
  • heroku config:add ROOT_URL=https://<nuttyserver>.herokuapp.com
  • git push heroku master
  • Thats it. Visit https://<nuttyserver>.herokuapp.com

Chrome Extension configuration:

  • open "chrome://extensions" in address bar
  • click "options" in the Nutty extension section
  • Set your new server domain name.
