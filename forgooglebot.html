Share linux/mac terminal over web using just a browser!

(Useful for remote support or pair-programming)

Installation steps

  • Click to install Nutty Extension.
  • Run 'bash install_host.sh' as root to install nutty.py and io.nutty.terminal.json
  • Click here (refresh Chrome if it doesn't work)

Demo

Details

Nutty uses WebRTC to share the terminal with peers in the browser. WebRTC is true peer-to-peer technology built into browsers to enable realtime communication between peers. All data transmitted using WebRTC is encrypted. Nutty uses WebRTC's DataChannel to share terminal peer-to-peer. Nutty uses PeerJS javascript library which provides abstraction over WebRTC APIs.

Nutty's web based terminal emulator (hterm) uses Chrome's Native Messaging API to communicate with the nutty.py script which creates pseudo terminals.

Browser supported: Chrome 29+. Firefox support coming soon!

Security notes

  • Communication between peers is direct p2p without any proxy and encrypted by WebRTC layer
  • Access to backend script nutty.py is restricted to nutty's chrome extension. Access to nutty's chrome extension is in turn restricted to https://nutty.io domain
  • Any terminal recorded and uploaded will not contain passwords as they are not echo'ed to the terminal
